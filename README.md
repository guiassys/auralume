# auralith
Auralith é um aplicativo desenvolvido para a criação de conteúdo digital no estilo lo-fi, inspirado por experiências imersivas como as da Lofi Girl. A plataforma permite a geração automatizada de músicas lo-fi, combinadas com uma identidade visual personalizável.

### Aelion

Aelion é o primeiro personagem criado dentro do Auralith.

Ele é um anjo solitário e contemplativo, frequentemente representado em cenários silenciosos e elevados, observando o mundo enquanto suas emoções se traduzem em música.

Aelion funciona como uma âncora emocional para o projeto, conectando o público à experiência sonora e visual.

---

## 🎯 Objetivo

Criar uma pipeline automatizada capaz de:

1. Gerar músicas lo-fi  
2. Criar cenas visuais coerentes  
3. Integrar personagem + ambiente + som  
4. Produzir vídeos prontos para publicação  
5. Distribuir conteúdo em plataformas como o :contentReference[oaicite:1]{index=1}  

---

## 🧠 Estratégia

O projeto utiliza modelos de linguagem (LLMs) e ferramentas como :contentReference[oaicite:2]{index=2} para:

- Gerar ideias musicais  
- Criar prompts visuais  
- Automatizar fluxos de criação  
- Integrar diferentes etapas do pipeline
